# 📘 Manual de Usuario - Control de Ventas SaaS

## Introducción
Bienvenido a Control de Ventas, la solución integral para la gestión de fuerza comercial, comisiones y liquidaciones.

## 🚀 Inicio Rápido

### Acceso
1. Ingrese a la URL de la aplicación.
2. Utilice las credenciales de Supabase asignadas por el administrador.
3. Si su cuenta aún no está activa, solicite acceso desde la pantalla de inicio de sesión.

## 📊 Dashboard
El panel principal ofrece una vista en tiempo real de:
- Ventas totales del mes.
- Comisiones acumuladas.
- Top colaboradores.
- Rendimiento por producto.

## 💼 Gestión de Ventas

### Registrar una Venta
1. Vaya a la sección **Ventas**.
2. Haga clic en **Nueva Venta**.
3. Complete el formulario (Cliente, Producto, Colaborador, Zona).
4. El sistema calculará automáticamente la comisión.
5. Guarde la venta.

### Filtros y Búsqueda
- Utilice la barra lateral para filtrar por estado, colaborador o fechas.
- Use la barra de búsqueda para encontrar ventas por cliente o DNI.

## 👥 Colaboradores
Administre su equipo comercial:
- **Altas y Bajas:** Registre nuevos comerciales o desactive cuentas.
- **Niveles:** Asigne niveles (Junior, Senior, Manager) con diferentes esquemas de comisiones.

## 💰 Liquidaciones
Genere pagos para su equipo:
1. Vaya a **Liquidaciones**.
2. Seleccione el periodo a liquidar.
3. Revise el cálculo automático.
4. Genere el PDF de liquidación para cada colaborador.

## ⚙️ Configuración
Personalice la aplicación:
- **Reglas de Comisión:** Defina porcentajes por producto o servicio.
- **Productos:** Administre su catálogo de servicios.
- **Usuarios:** Gestione los accesos al sistema.
