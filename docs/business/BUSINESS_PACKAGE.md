# 📊 Control de Ventas - Paquete de Venta

## 🎯 Resumen Ejecutivo

**Producto:** Sistema de Control de Ventas y Liquidaciones
**Tecnología:** React + Supabase + PWA
**Estado:** Funcional y operativo
**Mercado:** PYMEs con equipos comerciales

## 💻 Características Técnicas

### Stack Tecnológico
- **Frontend:** React 19.1.1 + Vite 7.1.6
- **UI:** Tailwind CSS + Lucide React
- **Backend:** Supabase (Auth + Database)
- **Almacenamiento:** Híbrido (Local + Cloud)
- **PWA:** Instalable en móvil/desktop

### Funcionalidades Core
- ✅ Gestión de ventas completa
- ✅ Control de colaboradores y comisiones
- ✅ Liquidaciones automáticas con IRPF/IVA
- ✅ Sistema de decomisiones por bajas
- ✅ Exportación Excel/PDF profesional
- ✅ Filtros avanzados y búsquedas
- ✅ Dashboard con métricas clave
- ✅ Autenticación dual (Supabase + Local)

### Arquitectura
- 📱 Responsive design
- 🔄 Sincronización automática
- 💾 Persistencia local para offline
- 🔐 Autenticación segura
- ⚡ Performance optimizada
